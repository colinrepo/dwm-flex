# Suckless.org介绍

[Suckless.org](https://dwm.suckless.org)是一个崇尚极简主义，致力于开发和维护“无用软件”（sucks less，意为“没那么糟糕”或“更少的垃圾”）的社区组织。他们的目标是创建比现有选项更精简、更高效、更可靠的软件。

知名项目包括:
+ **dwm**: 一款极简、动态窗口管理器，适用于 X Window 系统。
+ **st**: 一款简单的虚拟终端仿真器，作为 xterm 的替代品。
+ **dmenu**: 一款快速、轻量级的动态菜单，用于 X Window 系统，常用于启动程序。
+ surf: 一款基于 WebKit 的极简网页浏览器。
+ sbase: 一个旨在替代标准 Unix 工具集（coreutils）的简单实现。

# dwm-flex

Suckless提供的dwm方案是一个最小化的动态窗口管理方案，需要爱折腾的你去不断地探索各种插件的效果，然后去打补丁完善，逐渐改善才会好用。研究一个插件的效果需要先将该插件下载，然后对dwm打好补丁解决冲突等才能看到效果，而花了不少精力总算给一个插件融合到dwm源码了，一看效果，发现自己并不喜欢该效果，然后又得版本回退然后研究下一个插件，这需要不菲的时间和精力。

dwm-flex解决了suckless的dwm的上述痛点。dwm-flex基于dwm-flexipatch、st-flexipatch、dmenu-flexipatch。它将dwm所有的插件以宏编译的方式整合到了源码中，你只需要去调整配置文件中关于插件开启和关闭的宏，逐步去探索各插件的效果，喜欢的保持打开，不喜欢的关闭，最终就能得到一个你比较偏爱的dwm方案。效率极高节省了时间和精力又达到了想要的效果。
